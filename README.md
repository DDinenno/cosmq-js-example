# Cosmq JS Example

This is an example website for [Cosmq JS](https://github.com/DDinenno/cosmq-js)

## getting started

Clone this repo
> git clone git@github.com:DDinenno/cosmq-js-example.git

Install
> npm i

## Scripts

Run dev server 
> npm run dev

Create production build
> npm run build

Preview 
> npm run preview
